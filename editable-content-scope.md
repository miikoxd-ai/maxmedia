# Editable Content Scope

The redesign will shift the site toward a **modern black-and-red performance aesthetic** with content blocks that can be updated through a lightweight admin workflow.

| Asset or Area | Identified Subject | Planned Use |
| --- | --- | --- |
| `IMG_1568.JPEG` | A laptop screen showing photo-editing software with a blue supercar image being edited | Use as a supporting visual in a section about editing workflow, production process, or behind-the-scenes creative execution |
| `SNY09334.JPEG` | A portrait-style image of a young creator beside a red performance car | Use as the primary founder or personal-brand image in the hero or About section |
| `ferrarifinal.webp` | A close-up detail shot of a red Ferrari front end | Use as a bold feature image for the portfolio, showcase, or background accent area |

The editable backend should initially cover the highest-value website content rather than every visual detail.

| Editable Area | Fields |
| --- | --- |
| Hero section | Eyebrow, headline, supporting paragraph, primary CTA label, secondary CTA label |
| About section | Title, biography paragraph, credibility highlights |
| Portfolio section | Section title, intro copy, featured item titles, featured item descriptions, Instagram links |
| Contact section | Heading, support copy, email, phone, social links |
| Site settings | Brand name, footer text |

The first implementation should prioritize a **simple admin editing experience** over a full CMS. A small protected editor page backed by persistent storage is sufficient for this stage.
